About This Prototype

Evexia is a hackathon prototype demonstrating patient-controlled medical data aggregation. It allows patients to consolidate records from multiple Arizona hospitals, generate AI-powered summaries, and securely share specific data categories with healthcare providers.

Security & Privacy Features

  • Token-based sharing: Patients control access with expiring tokens
  • Scoped access: Providers only see authorized data categories
  • Access logging: All provider access is recorded
  • Data integrity: AI can flag but never delete your data
